Focusable items

Tab through the interactive elements. In desktop screenreaders you normally use the TAB key for this. With Shift + TAB you get the previous element. This also works in web browsers, which is a great way of quickly testing accessibility issues. (Read more)

Read elements

Experimental This one is a little bit trickier. In NVDA you start reading elements with "Insert + ↓ or Numpad +", VoiceOver uses "Configured VoiceOver Combination + A". Search for screenreader shortcut lists or search for reports of screenreader users to find out more about this. There are a lot of commands available for each screenreader software.
